## 🏗️ Arquitetura

O projeto segue o padrão **MVVM (Model-View-ViewModel)**, com a seguinte estrutura de pacotes:

```
app/
└── src/
    └── main/
        └── java/
            └── com.example.app/
                ├── data/         # Repositórios e fontes de dados
                ├── di/           # Injeção de dependências
                ├── ui/           # Telas e componentes visuais
                └── viewmodel/    # ViewModels
```

---

## 🛠️ Tecnologias e Bibliotecas

- **Linguagem:** Kotlin
- **UI:** Jetpack Compose
- **Navegação:** Navigation Component
- **Arquitetura:** MVVM
- **Build:** Gradle
